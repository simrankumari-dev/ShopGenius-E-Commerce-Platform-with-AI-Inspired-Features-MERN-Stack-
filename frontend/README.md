# OneCart – E-Commerce Web Application

OneCart is a modern e-commerce web application built using React and Vite.  
The project focuses on creating a clean, fast, and responsive online shopping experience using modern frontend technologies.

This application is part of an ongoing development process and demonstrates real-world e-commerce features such as user authentication, product browsing, and cart management.

---

## 🚀 Features
- User authentication (Google OAuth supported)
- Product listing with clean and responsive UI
- Add to cart functionality
- Toast notifications for user actions
- Client-side routing with React Router
- Mobile-friendly design
- Fast development and build using Vite

---

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Firebase
- Axios
- React Router DOM
- React Icons
- React Toastify
